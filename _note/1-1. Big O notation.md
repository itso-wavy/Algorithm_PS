# 1. 빅 😲 표기법

## (1) 정의

- 코드를 분류, 비교하고 성능을 평가하는 시스템
- `입력`과 `실행 시공간`의 관계를 함수로 나타낸 것
- 변화 추세를 거시적으로 바라본 것으로 생각하면 쉬움

## (2) 표기

```ts
O(f(n))

ex.
O(n+10) => O(n)
O(1000n+50) => O(n)
O(n²+5n) => O(n²)
```

## (3) 시간복잡도

① 산수(덧셈, 뺄셈, 곱셈, 나눗셈), 변수: `O(1)`

② 인덱스로 배열 탐색: `O(1)`

③ 루프 = 루프길이x연산 횟수: `O(n)`

④ 중복 루프 = 루프x루프: `O(n²)`

## (4) (보조)공간복잡도

- **시간(루프)과 무관**
- **데이터가 차지하는 공간에 주목할 것**

① 불리언, 숫자, undefined, null: `O(1)`

② string, 참조 타입 데이터(배열, 객체): `O(n)`

```ts
ex.
function func(n) {
  for(let i = 1; i <= n; i++) {
    console.log(i)
  }
}

시간복잡도: O(n)
공간복잡도: O(1)
```
